# 設計: 静的データファイル仕様テンプレート (data/)

## 概要

アプリケーションが参照する静的データファイル（マスタデータ）の仕様テンプレート。
`data/` ディレクトリに配置し、`tools/{domain}_tools.py` から読み込む。

## ファイル配置

`data/{data_name}.json`

---

## サンプルデータ

```json
{
    "{top_level_key}": [
        {
            "{field_1}": "値1",
            "{field_2}": "値2",
            "{field_3}": 100
        },
        {
            "{field_1}": "値3",
            "{field_2}": "値4",
            "{field_3}": 200
        }
    ]
}
```

## カスタマイズガイド

1. **データファイルの追加**: 業務要件に応じて `data/` 配下にJSONファイルを追加する
2. **Pydanticモデルの定義**: データ構造に対応するモデルを `models/data_models.py` に定義する
3. **読み込み関数の定義**: `tools/{domain}_tools.py` にデータ読み込み関数を定義する
