### 業務ルール一覧

| ルールID | ルール内容 | 適用タイミング | 例外条件 | 備考 |
|---|---|---|---|---|
| BRL-01 | 社員が入力した申請内容に基づき、社内申請ルールを参照して申請種別（交通費精算申請 または 経費精算申請）を判断しなければならない | 申請種別判断前 | なし | |
| BRL-02 | 申請種別の判断結果と根拠（参照した申請ルール）を社員に提示しなければならない | 申請種別判断後 | なし | |
| BRL-03 | 申請書作成に必要な情報が不足している場合、申請書作成前に対話で収集しなければならない | 申請書作成前 | 必要情報がすべて揃っている場合 | |
| BRL-04 | 申請書は収集した情報に基づいて生成しなければならない。収集していない情報を推測・補完してはならない | 申請書作成時 | なし | |
| BRL-05 | 生成した申請書の内容が申請ルールに準拠しているかをチェックしなければならない | 申請書作成後 | なし | |
| BRL-06 | 申請内容から申請種別（交通費精算申請 または 経費精算申請）を判断できない場合は、「交通費精算申請」「経費精算申請」の選択肢を社員に提示し、選択を促さなければならない | 申請種別判断後 | なし | |
| BRL-07 | 交通費精算申請は対象交通手段（電車・バス・タクシー・飛行機）の移動情報（移動日・出発地・目的地・交通手段・費用・業務目的）を一区間ずつ収集しなければならない | 情報収集時 | なし | |
| BRL-08 | 交通費精算申請の運賃は運賃データから自動計算しなければならない。電車は経路テーブル検索、バス・タクシー・飛行機は固定運賃を使用する | 申請書作成時 | なし | |
| BRL-09 | 交通費が10,000円を超える場合は事前に上長の承認が必要である旨を社員に通知しなければならない | 申請情報収集完了後 | 交通費が10,000円以下の場合 | |
| BRL-10 | 交通費精算申請において駅名を正規化して取り扱わなければならない（例：「渋谷駅」→「渋谷」） | 情報収集時 | なし | |
| BRL-11 | 経費精算申請は領収書画像から経費情報（購入日・店舗名・品目・金額）を品目ごとに自動抽出しなければならない | 情報収集時 | なし | |
| BRL-12 | 経費精算申請の経費区分（事務用品費・宿泊費・資格精算費・その他経費）は品目から自動判断しなければならない | 情報収集時 | なし | |
| BRL-13 | 経費が5,000円を超える場合は事前に上長の承認が必要である旨を社員に通知しなければならない | 申請情報収集完了後 | 経費が5,000円以下の場合 | |
| BRL-14 | 交通費精算申請・経費精算申請ともに、申請は経費発生日から3ヶ月以内でなければならない | 申請種別判断後 | なし | |
| BRL-15 | すべての申請に業務目的の記載が必要である | 情報収集時 | なし | |

### 判断基準表

| 判断ID | 判断項目 | 入力情報 | 判断条件 | 判断結果 | 次アクション |
|---|---|---|---|---|---|
| JD-01 | 申請種別の該当判断 | 社員が入力した申請内容テキスト、社内申請ルール | 申請内容が交通費精算申請または経費精算申請のいずれかの適用条件に明確に合致する | OK（申請種別特定） | 該当申請種別・申請書・申請先を社員に提示する |
| JD-02 | 申請種別の判断不能 | 社員が入力した申請内容テキスト、社内申請ルール | 申請内容から交通費精算申請・経費精算申請のいずれかを判断できない | NG（種別判断不能） | 「交通費精算申請」「経費精算申請」の選択肢を社員に提示し、選択を促す（BRL-06適用） |
| JD-03 | 申請情報の充足判断 | 収集済み申請情報、申請種別ごとの必須項目定義 | 必須項目がすべて収集済みである | OK（情報充足） | 申請書の自動作成を実行する |
| JD-04 | 申請情報の不足判断 | 収集済み申請情報、申請種別ごとの必須項目定義 | 未収集の必須項目が1件以上存在する | NG（情報不足） | 不足項目を特定し、対話で確認・収集する |
| JD-05 | 申請書ルール準拠チェック | 生成した申請書内容、申請ルール | 申請書の全項目が申請ルールの定義を満たす | OK（準拠） | チェック合格として社員に申請書を提示する |
| JD-06 | 申請書ルール不備チェック | 生成した申請書内容、申請ルール | 申請書の1項目以上が申請ルールの定義を満たさない | NG（不備あり） | 不備箇所と理由を社員に通知し、修正を促す |
| JD-07 | 申請期限チェック | 申請の経費発生日 | 経費発生日から申請日までが3ヶ月を超える | NG（期限超過） | 申請期限を超過している旨を社員に通知する |
| JD-08 | 交通費上長承認要否判断 | 収集済み交通費合計額 | 交通費合計が10,000円を超える | 要承認 | 上長の事前承認が必要である旨を社員に通知する（BRL-09適用） |
| JD-09 | 経費上長承認要否判断 | 収集済み経費合計額 | 経費合計が5,000円を超える | 要承認 | 上長の事前承認が必要である旨を社員に通知する（BRL-13適用） |
