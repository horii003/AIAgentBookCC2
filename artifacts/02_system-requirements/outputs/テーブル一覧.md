## 3. データ永続化の方針

| 対象データ | 永続化方式 | 参照先 |

| 判断ログ | ファイルストレージ（構造化データ） | DATA-007 |
| 社内申請ルール | ファイルストレージ（テキスト/ドキュメント） | DATA-001 |
| 申請書テンプレート | ファイルストレージ（Excel） | DATA-002 |
| 電車経路・運賃データ | ファイルストレージ（JSON） | DATA-003 |
| 固定運賃データ | ファイルストレージ（JSON） | DATA-004 |
